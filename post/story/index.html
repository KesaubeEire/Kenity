从CDPR谈起:踏实做事 & 故事永恒

2018-06-18 Plain Text Kesa

今天让我突然翻转的动机是Gamker的视频
曾经和Gamker有过一些不愉快
但是那都过去了,也是我的错
这次看Gamker的视频以及之后的事情
看结果,收获就这么铿锵的几句话:

  • 走出舒适区
  • 踏踏实实做事
  • 内容为王

走出舒适区

回到CDPR创业的年代,对比中国
其实是一模一样的
盗版横行 市场小众 经济低沉 社会萌芽
好多政治\社会\经济背景是类似的
但是就是走出了这么一个
这么一个在世界游戏史上
如此年轻但是如此别致的公司

原因其实很复杂
但是不管原因再怎么复杂
经过对比都可以发现
这不是很容易做到的
大家都是希望躺着能赚钱的
从一个发行商非要到成为一个游戏开发商
一定是很累 一定是很忙 一定有着理想
一定是需要 走出舒适区的
WX看到没有,是时候走出来了

踏踏实实做事

接续CDPR成功的原因其实很复杂
没有那么简单能get到所有类比和所有分歧的途径
但是如果非要总结的话
运气 + 实力 + 踏实的性格
是CDPR崛起的几个经典步骤
大概也是后起之秀逆袭的标准了吧

也是我的目标

运气: 代理博德之门成功拉动波兰游戏市场 + 拿下<巫师>的游戏版权
实力: 做出<<巫师>>这款永垂游戏历史的游戏
性格: CDPR最令人称道的”波兰蠢驴”的 不防盗版_免费DLC_扩展包可以叫巫师4 的反商业的商业行为

内容为王

回忆<巫师>为什么做得这么好
为什么给我创造了那么多感动的回忆
谈及CDPR为什么总是容易是我热泪盈眶
为什么

“CDPR,或者说<巫师>没有任何创新”
“但是他们把RPG做到了极致”
–GCores

游戏系统会被忘记\法术会被遗忘\所有曾经体会过的东西都战胜不了时间的变迁
我只留下了对那个游戏世界的眷恋和对那段故事的怀念
但是
这一切的一切都是因为
CDPR对这个游戏内容的精彩控制
说到底,内容即是故事

故事,是生活的比喻

// 让我又想起了那段话,想起来推荐这本书给我的木鱼  
// 啊哈  
// 我多么希望能够再次回到  
// 能够回到之前的时光,之前没有荒废所有的时候  
// 呵呵,我这XJB的完美拖延症  
// 但是...我还是自己SB吖

一切都绕了回来,故事,真的是非常非常重要的消费品
一个抽象的消费品
但是确实是我们这种拥有智慧的生物始终需要始终贪婪着的
然后又看到一篇有关<<这是我的战争>>发行商
同时又是最最原初<<巫师>>游戏的开发者的公司的一篇报道
这家公司不是CDPR
我深刻的知道,做好故事,做好内容的必要性了
now, a little bit nap
start your Business!!!!!
